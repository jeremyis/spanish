## Present Subjunctive mood

- use when the perspective of the subject conveys **doubt or uncertainty**.
  - **Mario dudas que** la persona **confirme** su reservación.
  - Mario **doubts** that the person will confirm his reservation.

- **main clause**: (or independent clause) is the clause in indicative mood.
- **dependent clause**: (or noun clause) clause containing verb in subjunctive mood. 
  - noun clause because it functions similar to that of a noun in a simple sentence.

### Usage

  - Both a main and a dependent clause, separated by **que**.
  - Clauses mut be about different subjects.

### Formation 

1. Drop the o in the yo indicative form
2. Add the corresponding conjugation

|                         | ar   | er/ir |
|-------------------------|------|-------|
| yo                      | e    | a     |
| tu                      | es   | as    |
| el / ella / usted       | e    | a     |
| nosotros                | emos | amos  |
| ellos / ellas / ustedes | en   | an    |

### Spelling changes

- NOTE: These are to maintain "c" and "g" sounds when the normal conjugation flow appends an e or i
  - recall: "c"s and "g"s are pronounced different when followed by e or i

### More usage

1. To reflects denial, doubt, disbelief or uncertainty about relaity / outcome of actions of different subject.

  | verb               | meaning        |
  |--------------------|----------------|
  | dudar              | to doubt       |
  | negar              | to deny        |
  | no creer           | not to beleive |
  | no estar seguro de | not to be sure |
  | no pensar          | not to think   |

2. Following verbs that express advice, command, or suggestion.

  | verb       | meaning                                      |
  |------------|----------------------------------------------|
  | aconsejar  | to advice                                    |
  | decir      | to tell (with intention of giving a command) |
  | dejar      | to allow                                     |
  | exigir     | to demand                                    |
  | hacer      | to make                                      |
  | ordenar    | to order                                     |
  | recomendar | to recommend                                 |
  | sugerir    | to suggest                                   |

3. To communicate attitudes, feelings, or emotions in the dependent clause about a different subject.

  | verb              | meaning        |
  |-------------------|----------------|
  | alegrar(se) de    | to be glad     |
  | enojarse de       | to be angry    |
  | estar contendo de | to be glad     |
  | molestar(se) que  | to get annoyed |
  | sentir            | to regret      |
  | temer             | to fear        |

4. To express hope, wish, or preference about the actions of subjects in the dependent clause.

  | verb        | meaning   |
  |-------------|-----------|
  | desear      | to wish   |
  | esperar     | to hope   |
  | insistir en | to insist |
  | preferir    | to prefer |
  | querer      | to want   |

5. To ask permission or make a request.

  | verb     | meaning  |
  |----------|----------|
  | pedir    | to ask   |
  | rogar    | to beg   |
  | suplicar | to plead |

6. To communicate surprise or disbelief

  | verb             | meaning         |
  |------------------|-----------------|
  | maravillar de    | to marvel       |
  | sorpender(se) de | to be surprised |

  * Note sorpender can be impersonal: nos sorpende (it surprises us), or reflexive: nos sorpendemos (we are surprised)

7. After impersonal expressions that express disbelief, doubt, uncertainity, necessity, etc...

  | phrase                    | meaning             | - | phrase           | meaning           |
  |---------------------------|---------------------|---|------------------|-------------------|
  | conviene                  | it is advisable     |   | es necesario     | it is necessary   |
  | es dudoso                 | it is doubttful     |   | es posible       | it is possible    |
  | es importante, es preciso | it is important     |   | es probable      | it is probable    |
  | es impossible             | it is impossible    |   | es una pena      | it is a pity      |
  | es increíble              | it is unbelieveable |   | es útil          | it is useful      |
  | es inútil                 | it is useless       |   | más vale         | it is better      |
  | es lástima                | it is a pity        |   | ojalá            | I wish, if only   |
  | es mejor                  | it is better        |   | parece (mentira) | it seems (unreal) |

#### Adverb clauses

8. Conjunctions and phrases: if anticipation, doubt, purpose or a condition concerning the outcome of an action or event is implied:

  | phrase             | meaning                | - | phrase         | meaning                |
  |--------------------|------------------------|---|----------------|------------------------|
  | a condición de que | provided that          |   | con tal que    | provided that          |
  | a fin de que       | in order that, so that |   | en caso de que | in case (that)         |
  | a menos que        | unless                 |   | para que       | in order that, so that |
  | antes (de) que     | before                 |   | sin que        | without                |

9. 
  - Used *after* certain expressions when the event has not yet occured or is expected to occur after the main clause
  - there is uncertainty that the event will take place

  A. Convey time of the action:

  | phrase           | meaning        | - | phrase          | meaning    |

  | después (de) que | after          |   | tan pronto como | as soon as |
  | en cuanto        | as soon as     |   | una vez que     | once       |

## Present Perfect Subjunctive mood


- Formation: Present subjunctive of haber with past participio: hayan balando
- Useage: 
  - Subjunctive useage, but when dependent clause happened before main clause
  - Can also be if the main clause is in the future and dependent is now.
